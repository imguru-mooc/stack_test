스택 동작 시각화 연습 도구

숫자를 입력해 push 하고, pop 버튼으로 꺼낸다. 스택의 맨 위(top)는 항상 강조되어 보인다.

스택의 C 구현

배열 기반 · 오른쪽 도구와 같은 규칙 (top 초기값 −1)

/* 배열 기반 스택 */int stack[MAX]; /* 값 저장 배열 */int top = -1; /* 비어 있으면 top = -1 */ int is_empty(void) { return top == -1; }int is_full(void) { return top == MAX-1; } /* push: top을 올리고 그 자리에 저장 */void push(int x) { if (is_full()) return; /* overflow */ stack[++top] = x; /* top: 0,1,2,... */} /* pop: top 값을 꺼내고 top을 내림 */int pop(void) { if (is_empty()) return -1; /* underflow */ return stack[top--]; /* 반환 후 감소 */} /* 아래 순서대로 실행된다 */int main(void) { push(2); push(7); pop(); push(5); push(1); pop(); pop(); return 0;}
느리게 빠르게

‘main 실행’은 위 시나리오대로 오른쪽 스택을 자동으로 움직입니다. ‘다음 줄’로 한 줄씩 짚어가며 볼 수도 있습니다. 실행 중인 main 줄은 초록, 해당 push/pop 구현 줄은 주황으로 강조됩니다.
